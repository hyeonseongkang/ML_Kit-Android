# ML_Kit-Android

## ML Kit란
머신 러닝에 대한 깊은 지식 없이도 Android 및 ios 앱에서 머신 러닝 모델을 사용할 수 있게 해주는 모바일 SDK이다.

<br>

## ML Kit 제공 API

1. 얼굴 감지

2. 텍스트 인식

3. 바코드 스캔

4. 이미지 라벨

5. 랜드마크 인식
